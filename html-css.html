HTML

  • What?
    1. HyperText Markup Language.
    2. Hypertext is text displayed on a computer display or other electronic devices.
  • Why?
    1. Proper formatting of text and images for your Internet browser.
    2. Tell your web browser what each part of a website is
    3. Structure or Layout of the page
2

CSS

  • What?
    1. Cascading Style Sheets.
    2. "Cascading" means that styles can fall (or cascade) from one style sheet to another, enabling multiple style sheets to be used on one HTML document. (Cascade জলপ্রপাত).
  • Why?
    1. For Style Structure or Layout.
    2. Control The HTML / How / Where show?
